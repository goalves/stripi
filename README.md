# Stripi

Stripi is yet another Stripe Elixir API.
This is still in early stages of development.

- [x] Charges
- [x] Customers
- [x] Ephemeral Keys
- [ ] Balance
- [ ] Disputes
- [ ] Events
- [ ] File Uploads
- [ ] Payouts
- [ ] Refunds
- [ ] Tokens
- [ ] Sources
- [ ] Subscriptions
- [ ] Connect


## Installation

The package can be installed by adding `Stripi` to your list of dependencies in
`mix.exs`:

```elixir
def deps do
  [
    {:stripi, "~> 0.1.0"}
  ]
end
```
